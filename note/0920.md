### 持久化補充
* [docker volumes](https://myapollo.com.tw/zh-tw/docker-volumes/)
* 創建`docker volume create myvol1`

* `docker run -it -v myvol1:/data busybox sh`，如果沒有/data會自動創建

* 另一種掛載方式`docker run -it --name d3 --volumes-from d2 busybox sh`
* 一次掛載多數
